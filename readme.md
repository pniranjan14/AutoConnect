# 🚗 AutoConnect – Vehicle Service Automation

AutoConnect is a full-stack web application that automates vehicle service booking and management by connecting users with nearby service centers. The platform provides a seamless experience for customers, service providers, and administrators through role-based dashboards and real-time service tracking.

---

## 📌 Features

### 👤 User Module
- Register and login securely
- Find nearby service centers
- Book vehicle services
- Track booking status
- Emergency assistance requests
- Submit reviews and ratings

### 🏪 Service Provider Module
- Register and manage service center profile
- Add and manage services with pricing
- View and respond to booking requests
- Manage service status (Pending, Accepted, Completed)
- View analytics and customer feedback

### 🛡️ Admin Module
- Approve or reject service providers
- Manage users and service centers
- Monitor system activities
- Ensure quality and authenticity
